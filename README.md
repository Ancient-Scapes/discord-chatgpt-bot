# discord-chatgpt-bot

Discord Botを通してChatGPT APIを実行し、チャンネルに投稿します。
`npm run dev` をローカルで実行中にBotが反応します。

## Environments

以下が必要です。

- DiscordのBot Token
- OpenAIのAPI KEY

.env

```zsh
BOT_USER_NAME=Botユーザ名
BOT_TOKEN=xxx
API_KEY=xxx
```

## Usage

```zsh
npm i
npm run dev
```
